研究方向 1	基于 LLM（大语言模型）的 RTL 代码生成	基于 CodeLlama、Qwen-Code 等大语言模型，优化 RTL（Verilog/VHDL）代码生成范式，适配硬件设计场景的 prompt 工程，验证生成代码的语法正确性与功能完整性	集成电路设计、硬件加速、课程设计 / 科研项目	1. 贴合 AI + 硬件设计前沿方向，研究价值高；2. 可复现性强，开源 LLM 模型与数据集丰富；3. 能落地为实用工具，适配工程场景
研究方向 2	Mg₃Bi₂基柔性热电半导体	研究 Mg₃Bi₂基热电材料的柔性化制备工艺，分析温度、应力对热电转换效率的影响，搭建柔性热电器件性能测试平台	新能源材料、柔性电子、实验类科研项目	1. 聚焦低维热电材料前沿领域，应用场景广；2. 实验数据可量化，便于文献计量学分析；3. 开源仿真工具与数据集可支撑研究
研究方向 3	RISC-V CPU 设计与优化	基于 RISC-V 指令集架构，设计轻量级 CPU 核，优化流水线、存储架构与功耗，适配 FPGA 原型验证与 ASIC 流片前仿真	开源芯片、嵌入式系统、硬件竞赛 / 课程设计	1. RISC-V 生态开源且活跃，资料易获取；2. 可从基础核扩展至多核 / 异构系统，研究延展性强；3. 符合国产芯片自主化发展趋势
开源项目 1	CodeLlama-RTL	基于 Meta CodeLlama 模型适配 RTL 代码生成的开源项目，包含硬件设计专用 prompt 模板、RTL 代码校验脚本与测试用例库	基于 LLM 的 RTL 代码生成研究	1. 专为硬件设计场景优化，开箱即用；2. 支持 Verilog/VHDL 双语言生成，适配主流 EDA 工具；3. 开源协议宽松，可自由二次开发
开源项目 2	ThermoElectric-Materials	热电材料性能仿真开源库，支持 Mg₃Bi₂、Bi₂Te₃等主流热电材料的输运特性计算，包含柔性基底应力仿真模块	Mg₃Bi₂基柔性热电半导体研究	1. 覆盖热电材料核心仿真需求，数据与实验结果可对标；2. 基于 Python 生态开发，易与数据分析工具集成；3. 文档完善，含材料参数数据库与可视化脚本
开源项目 3	PicoRV32	极简 32 位 RISC-V CPU 核，Verilog 实现，支持 RV32IM 指令集，资源占用极低，适配 Xilinx/Intel 等主流 FPGA	RISC-V CPU 设计研究	1. 代码量小（约 2k 行 Verilog），易读易改；2. 包含完整测试用例与 FPGA 移植 Demo；3. 广泛用于教学与科研项目，社区支持完善
